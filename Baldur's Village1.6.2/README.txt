
INSTALLATION GUIDE
----------------------------------------------------------------
Requirements:
This mod requires SMAPI﻿﻿, Content Patcher,﻿ and Portraiture﻿ to be installed.
You must move the BVportrait files to the Portraiture‘s mod folder. BVportrait file path: BaldursVillage 1.0.0 ->BVportraits
----------------------------------------------------------------
0. Prerequisites (Important!)
- SMAPI﻿﻿
﻿- Content Patcher﻿﻿
- Portraiture﻿ Extract the folder and move it to: Steam -> Common -> Stardew Valley -> Mods

1. Installing Baldur‘s Village
﻿- On main files page, downloading the Baldur‘s Village folder, extract the folder and move it to: Steam -> Common -> Stardew Valley -> Mods
- Also ﻿downloading ﻿BVportraits ﻿folder, ﻿﻿extract the ﻿BVPortraits folder and move it to: Steam -> Common -> Stardew Valley -> Mods -> Portraiture -> Portraits

2. Launching the Mod
Start the game using SMAPI:﻿ Steam -> Common -> Stardew Valley -> StardewModdingAPI

3. Changing Portraits In-Game
While interacting with a character, press P to change the portrait.




安装指南
----------------------------------------------------------------
必需条件：
此模组需要安装 SMAPI、Content Patcher 和 Portraiture。
你必须将 BVportrait 文件移动到 Portraiture 模组的文件夹中。
BVportrait 文件路径：BaldursVillage 1.0.0 -> BVportraits
----------------------------------------------------------------
0. 先决条件（重要！）
- SMAPI
- Content Patcher
- Portraiture (请解压 Portraiture 文件夹并移动至以下路径：Steam -> Common -> Stardew Valley -> Mods)

1. 安装 Baldur’s Village
- 在主要文件页面下载 Baldur’s Village 文件夹，解压后移动到以下路径：Steam -> Common -> Stardew Valley -> Mods
- 另外，下载 BVportraits 文件夹，解压 BVPortraits 文件夹，并将其移动到以下路径：Steam -> Common -> Stardew Valley -> Mods -> Portraiture -> Portraits

2. 启动模组
使用 SMAPI 启动游戏：Steam -> Common -> Stardew Valley -> StardewModdingAPI

3. 游戏内更换肖像
与角色互动时，按P键即更换成完整肖像。
